Salesforce Pack
---------------

The Salesforce Community Pack gives visibility into your salesforce login, loginAs and logout events. It provides out of the box tag, saved queries and dashboards that allow you to easily track user and admin access.

This Pack contains the following:

* Saved Queries: Saved queries that allow for the tracking and analysts of user access.
* Tags and Alerts: Tags that highlight logins, logouts and loginAs events.
* Dashboards: A basic dashboard that records total number of logins, logouts and loginAs events.
